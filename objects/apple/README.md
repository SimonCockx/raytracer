De bijgevoegde normal map is een ingebakken versie van de bijgevoegde 
originele bump map (ingebakken via Blender). De normalen zijn opgeslagen 
met hun object-space XYZ componenten in de RGB kanalen van de normal map. 
In principe zouden de bump en normal map dus beiden dezelfde shading moeten 
opleveren.

Opmerking over de shading:
Zoals je kan merken aan het texture bestand, zit hier al behoorlijk wat 
belichting in 'ingebakken'. Er zijn zelfs duidelijk speculaire highlights 
te zien! In die zin is het dus niet echt een goede texture (wat normaal 
gezien de diffuse kleur van het object moet weergeven, onafhankelijk van 
wat voor belichting er op het object valt)!
